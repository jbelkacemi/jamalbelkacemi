Windows Server Evaluation to Full

Convertir Windows Server evaluation a versión completa sin reinstalar el sistema. Proceso directo usando DISM o slmgr.

Verificar Estado Actual

# Ver edición actual
Get-WindowsEdition -Online

# Ver ediciones disponibles
DISM /Online /Get-TargetEditions

# Estado de licencia
slmgr /dli

Conversión con slmgr

Método alternativo solo para cambiar la clave de producto.

# Instalar clave
slmgr /ipk XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

# Activar
slmgr /ato

# Verificar
slmgr /xpr

Verificación Final

# Verificar conversión
Get-WindowsEdition -Online
slmgr /dli
slmgr /xpr

Solución de Errores

# Error 0x80070490
sfc /scannow
DISM /Online /Cleanup-Image /RestoreHealth

# Problemas KMS
slmgr /skms servidor-kms:1688
slmgr /ato
